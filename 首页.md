# 首页 Session 模块功能区分

本文档记录了首页（HomeView.vue）中各个 Session 模块的功能、用途及技术实现细节。

## Session 1: 英雄区 (Hero Section)

*   **位置**: 页面顶部
*   **主要功能**: 视觉冲击、品牌展示、核心引导
*   **内容**:
    *   全屏背景视频 (`src/assets/videos/hero.mp4`)
    *   遮罩层 (Overlay) 用于增强文字对比度
    *   主标题：“智启未来·科技之美”
    *   副标题：“以现代、极简的设计语言，呈现电影级光影质感”
    *   行动按钮 (CTA): “立即探索”
*   **技术特点**:
    *   使用 `hero-fullbleed` 实现 100vw 全宽破框布局
    *   采用 `height: 100vh` 配合负边距实现通顶全屏效果
    *   视频自动播放、静音、循环

## Session 2: 未来功能探索 (Content Placeholder)

*   **位置**: 英雄区下方
*   **主要功能**: 内容占位、过渡区域
*   **内容**:
    *   标题：“未来功能探索”
    *   描述文案
    *   骨架屏 (Skeleton) 占位演示
*   **技术特点**:
    *   垂直居中布局
    *   **视觉风格**: 黑色背景，白色文字

## Session 3: 横向滚动展示 (Horizontal Scroll Section)

*   **位置**: 页面中部
*   **主要功能**: 项目/产品卡片展示
*   **内容**:
    *   多张图文卡片横向排列
    *   每张卡片包含图片（4:5 比例）、标题、描述
*   **技术特点**:
    *   **交互方式**: 鼠标滚轮控制横向滚动 (GSAP ScrollTrigger)
    *   **视觉风格**: 黑色背景，白色文字
    *   **布局**: 100vw 全宽，卡片精确对齐屏幕边缘
    *   **组件**: `HorizontalScrollSection.vue`

## Session 4: 双排自动滚动 (Double Scroll Section)

*   **位置**: 页面底部
*   **主要功能**: 名言/评价动态展示
*   **内容**:
    *   标题区：主标题“社区回响”，副标题“倾听来自全球开发者的灵感与洞见，汇聚科技与人文的思考”
    *   两排卡片，反向自动滚动
    *   卡片内容：名言引用、作者名、用户头像（正方形）
*   **技术特点**:
    *   **动画**: CSS Keyframes 实现无限循环滚动 (Marquee效果)
    *   **交互**: 鼠标悬浮时当前行暂停，卡片高亮上浮
    *   **视觉风格**: 黑色背景，长方形卡片，极简设计
    *   **组件**: `DoubleScrollSection.vue`

## Session 5: 隐私与安全 (Privacy Section)

*   **位置**: 页面底部 (倒数第二)
*   **主要功能**: 强调数据安全与隐私保护承诺
*   **内容**:
    *   标题区：主标题“隐私与安全”，副标题“我们始终将您的数据安全与隐私保护放在首位...”
    *   三张特性卡片：端到端加密、零知识证明、合规与审计
    *   卡片样式：左对齐粗体小标题 + 正文
*   **技术特点**:
    *   **视觉风格**: 黑色背景，白色文字，极简长方形卡片
    *   **布局**: Flex 响应式布局（桌面端三列，移动端单列）
    *   **交互**: 鼠标悬浮时卡片上浮、高亮
    *   **组件**: `PrivacySection.vue`

## Session 6: 理念过渡 (IdeaSpark Section)

*   **位置**: 页面最底部
*   **主要功能**: 品牌理念传达，视觉过渡
*   **内容**:
    *   超大左对齐标题：**IDEASPARK**
    *   灰色副标题：描述创意激发与工具赋能
    *   **底部区域**:
        *   灰色细分割线
        *   左侧：Logo 占位图标 + 版权信息 (© 2025 IDEASPARK)
        *   右侧：多列导航链接（条款、资源、链接），采用粗体标题+灰色列表项
*   **技术特点**:
    *   **视觉风格**: 黑色背景，高对比度文字
    *   **排版**: 极简主义左对齐，强调文字排印张力
    *   **布局**: Flex 响应式布局，支持底部栏多列自适应
    *   **组件**: `IdeaSparkSection.vue`

## Session 7: 视觉冲击 (Impact Section)

*   **位置**: 页面最底层 (Fixed Reveal)
*   **主要功能**: 视觉冲击，品牌强化，反向揭示效果
*   **内容**:
    *   全屏绿色背景 (#4ADE80)
    *   超大文字 "IDEASPARK" 撑满屏幕
*   **技术特点**:
    *   **Fixed Reveal**: 使用 `position: fixed` 和 `z-index` 实现底部固定，上层内容滚动揭示的效果（从下往上看到的错觉）。
    *   **Responsive Text**: 使用 `vw` 单位和媒体查询适配文字大小，确保视觉冲击力。
    *   **组件**: `ImpactSection.vue`
