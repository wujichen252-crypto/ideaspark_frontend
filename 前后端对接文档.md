# IdeaSpark 完整接口文档

## 项目概述

IdeaSpark 是一个创新项目管理平台，提供用户管理、团队协作、项目市场、社区交流等功能。本文档详细描述了后端 API 接口，方便前端开发者进行对接。

## 技术栈

- **后端**：Spring Boot 3.4.1 + Java 17 + JPA + MySQL
- **认证**：JWT
- **文件存储**：阿里云 OSS
- **接口风格**：RESTful

## 基础信息

- **API 基础路径**：`http://47.108.232.238:9000/api`
- **认证方式**：Bearer Token（JWT）
- **响应格式**：统一 JSON 格式

### 统一响应格式

```json
{
  "status": 200,
  "message": "操作成功",
  "data": {}
}
```

| 字段    | 类型         | 说明        |
| ------- | ------------ | ----------- |
| status  | number       | HTTP 状态码 |
| message | string       | 响应消息    |
| data    | object/array | 响应数据    |

### 分页响应格式

```json
{
  "status": 200,
  "message": "获取成功",
  "data": {
    "list": [],
    "total": 100,
    "page": 1,
    "size": 10
  }
}
```

## 认证机制

所有需要认证的接口都需要在请求头中添加 `Authorization` 字段：

```
Authorization: Bearer {token}
```

## 详细接口文档

### 1. 用户管理

#### 1.1 用户登录

**接口**：`POST /api/user/login`

**描述**：用户通过邮箱和密码登录系统，获取 JWT Token

**请求参数**：

| 字段     | 类型   | 必填 | 描述     |
| -------- | ------ | ---- | -------- |
| email    | string | 是   | 用户邮箱 |
| password | string | 是   | 用户密码 |

**请求示例**：

```json
{
  "email": "user@example.com",
  "password": "password123"
}
```

**响应示例**：

```json
{
  "status": 200,
  "message": "登录成功",
  "data": {
    "token": "eyJhbGciOiJIUzI1NiIs...",
    "user": {
      "id": 1,
      "username": "张三",
      "email": "user@example.com",
      "avatar": "https://example.com/avatar.jpg"
    }
  }
}
```

**错误码**：

- `400`：参数错误
- `401`：邮箱或密码错误
