# 项目市场页面布局文档 (Project Market Page Layout)

本文档详细描述了 `MarketView.vue` 页面的宏观布局、组件结构及样式层级，旨在帮助开发者快速理解页面构成。

## 1. 宏观布局 (Macro Layout)

页面整体采用垂直流式布局，主要分为两大区域：
1.  **Market Header (头部区域)**：包含顶部操作栏（筛选/搜索）和 Hero Card（核心展示区）。
2.  **Market Content (内容区域)**：包含精选创作者列表、项目列表及分页。

页面容器 `.market-container` 撑满最小高度 `100vh`，背景色为浅灰 `#f6f5f7`。

---

## 2. 详细结构 (Detailed Structure)

### 2.1 Market Header (头部区域)
**位置**: 页面顶部，背景色为白色 `#ffffff`，底部有 1px 边框。
**内边距**: `padding: 84px 20px 24px` (顶部留出空间适配 fixed AppHeader)。

内部包含 `.header-content` (最大宽度 1280px，居中)，子模块如下：

#### A. Top Actions (顶部操作区)
位于 Hero Card 上方，采用 Flex 布局 `justify-content: space-between`。
-   **左侧 - Header Tabs (分类筛选)**:
    -   组件: `n-tabs` (Naive UI)
    -   样式: `size="large"`, `type="line"`
    -   选项: 全部、热门、最新、Web应用、移动端、设计资源。
-   **右侧 - Search Bar (搜索栏)**:
    -   组件: `n-input-group` + `n-input` + `n-button`
    -   宽度: `400px`
    -   功能: 支持回车或点击按钮搜索。

#### B. Hero Card (核心展示卡片)
位于 Top Actions 下方，是一个醒目的横幅区域。
-   **样式**: 渐变背景 + 网格纹理 + 动态 Blob 动画。
-   **布局**: Flex 布局，左右分栏。
-   **内容**:
    -   **Hero Text (左侧文本)**:
        -   Badge: "AI 创意引擎 2.0 已上线" (带脉冲动画)。
        -   Title: "发现无限创意..." (H1, 渐变高亮)。
        -   Subtitle: 描述文案。
        -   Actions: "开始探索" (Primary Button) + "我是创作者" (Ghost Button)。
    -   **Hero Visual (右侧视觉)**:
        -   包含磨砂玻璃卡片 (`.glass-card`) 的装饰性元素，仅在大屏 (`min-width: 1024px`) 显示。

---

### 2.2 Market Content (内容区域)
位于 Header 下方，最大宽度 1280px，居中。
**内边距**: `padding: 24px 20px 32px`。

#### A. Creators Section (精选创作者)
-   **Title**: H2 "精选创作者"。
-   **List Container**: 横向滚动容器。
    -   **Scroll Buttons**: 左右悬浮箭头按钮，Hover 时显示。
    -   **Creators List**:
        -   **Creator Card**: 卡片样式，包含背景图 (`bg`) 和头像/昵称 (`creator-info`)。
        -   交互: 悬浮上浮效果。

#### B. Project List Header (项目列表头)
-   **布局**: Flex `justify-content: space-between`。
-   **左侧**: Title "精选项目" + Count "共 X 个项目"。
-   **右侧**: Sort Dropdown (排序下拉框)。
    -   组件: `n-select`
    -   选项: 默认排序、最新发布、最多点赞、最多浏览。

#### C. Project Grid (项目网格)
-   **布局**: `n-grid` (Naive UI)。
-   **响应式列数**:
    -   Mobile: 1 列
    -   Small: 2 列
    -   Medium/Large: 3 列
-   **间距**: X/Y Gap 24px。
-   **Project Card (项目卡片)**:
    -   **Thumbnail**: 封面图，悬浮显示“喜欢”按钮。
    -   **Body**: 标题、作者信息 (Avatar + Name)、点赞数、标签 (Tags)。

#### D. Pagination (分页)
-   **位置**: 列表底部居中。
-   **组件**: `n-pagination`。

## 4. 关键样式说明 (Key Styles)

-   **响应式**: 主要依赖 `max-width: 1280px` 限制内容宽度，以及 `n-grid` 的响应式属性 `responsive="screen"`。
-   **主题色**:
    -   Primary: `#4ade80` (绿色系)
    -   Background: `#f6f5f7`
    -   Card Background: `#ffffff`
-   **动画**:
    -   Hero Card 背景使用了 CSS Animation (`float`, `pulse`)。
    -   卡片 Hover 效果使用了 `transform: translateY` 和 `box-shadow`。
